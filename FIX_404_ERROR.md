# 🔧 حل خطأ 404 على Vercel - دليل سريع

## المشكلة

الموقع يظهر خطأ **404 - NOT FOUND**

هذا يعني أن Vercel لم يجد ملف `index.html` الرئيسي.

---

## 🎯 السبب

الملفات لم تُرفع بشكل صحيح إلى GitHub. ربما:
- ✗ الملفات موجودة في مجلد فرعي
- ✗ الملفات لم تُرفع بالكامل
- ✗ اسم الملف غير صحيح (مثل Index.html بدلاً من index.html)

---

## ✅ الحل السريع (5 دقائق)

### الخطوة 1: تحقق من GitHub

```
1. اذهب إلى https://github.com/YOUR_USERNAME/taj-guides
2. انقر على "Code"
3. تأكد من أن الملفات موجودة:
   - index.html ✓
   - styles.css ✓
   - script.js ✓
   - guides/ (مجلد) ✓
```

### الخطوة 2: إذا كانت الملفات ناقصة

**استخدم GitHub Desktop:**

```
1. افتح GitHub Desktop
2. اختر المستودع "taj-guides"
3. انقر "Show in Explorer"
4. انسخ جميع ملفات الموقع إلى هذا المجلد:
   - index.html
   - styles.css
   - script.js
   - guides/ (المجلد)
   - جميع الملفات الأخرى
5. في GitHub Desktop:
   - ستظهر الملفات الجديدة
   - أدخل رسالة: "Add all website files"
   - انقر "Commit to main"
   - انقر "Push origin"
```

### الخطوة 3: أعد النشر على Vercel

```
1. اذهب إلى https://vercel.com/dashboard
2. اختر مشروع "taj-guides"
3. اذهب إلى "Deployments"
4. انقر على أحدث deployment
5. انقر "Redeploy"
6. انتظر 1-2 دقيقة
```

### الخطوة 4: اختبر الموقع

```
1. افتح https://taj-guides.vercel.app
2. يجب أن يحمّل الآن بدون خطأ
3. اختبر جميع الصفحات
```
