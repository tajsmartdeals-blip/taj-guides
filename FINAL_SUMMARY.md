## 🚀 البدء السريع

### الخطوة 1: تحضير البيئة

```bash
# الذهاب إلى مجلد المشروع
cd /home/ubuntu/taj-guides-website

# إنشاء مستودع Git
git init
git add .
git commit -m "Initial commit: Taj Guides Website"
```

### الخطوة 2: إنشاء حسابات مجانية

1. **GitHub** - https://github.com
2. **Vercel** - https://vercel.com
3. **Google Analytics** - https://analytics.google.com
4. **Gmail** - https://mail.google.com

### الخطوة 3: النشر على Vercel

```bash
# رفع إلى GitHub
git remote add origin https://github.com/YOUR_USERNAME/taj-guides.git
git push -u origin main

# ثم اتبع: VERCEL_DEPLOYMENT.md
```

### الخطوة 4: إعداد النطاق (اختياري)

1. اشتر نطاق من Namecheap أو GoDaddy
2. أضفه إلى Vercel
3. اتبع إعدادات DNS من Vercel

### الخطوة 5: إضافة Google Analytics

1. اتبع: ANALYTICS_SETUP.md
2. أضف كود التتبع إلى HTML
3. راقب الزيارات
