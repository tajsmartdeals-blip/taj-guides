# 🚀 دليل شامل: إنشاء حساب GitHub ونشر على Vercel

## المرحلة الأولى: إنشاء حساب GitHub

### الخطوة 1: الذهاب إلى GitHub

```
اذهب إلى: https://github.com
```

### الخطوة 2: انقر على "Sign up"

```
ستجد زر "Sign up" في أعلى اليمين
```

### الخطوة 3: أدخل بيانات التسجيل

```
1. البريد الإلكتروني: tajsmartdeals@gmail.com
2. كلمة المرور: اختر كلمة قوية (مثل: Taj@2024Secure!)
3. اسم المستخدم: taj-guides (أو أي اسم تفضله)
4. اختر "I prefer not to receive" للبريد الإعلاني
```

### الخطوة 4: التحقق من البريد الإلكتروني

```
1. افتح بريدك الإلكتروني
2. ستجد رسالة من GitHub
3. انقر على رابط التحقق
4. تم! ✅ حسابك مفعّل
```

### الخطوة 5: إكمال الملف الشخصي

```
1. أدخل اسمك الحقيقي (اختياري)
2. أدخل موقعك (اختياري)
3. أدخل السيرة الذاتية (اختياري)
4. انقر "Continue"
```

### الخطوة 3: إنشاء المستودع

```
انقر "Create repository"
تم! ✅ المستودع جاهز
```

## المرحلة الثالثة: رفع ملفات الموقع

### الطريقة 1: استخدام GitHub Desktop (الأسهل)

#### الخطوة 1: حمّل GitHub Desktop

```
اذهب إلى: https://desktop.github.com
حمّل التطبيق
ثبّته على جهازك
```

#### الخطوة 2: سجل الدخول

```
1. افتح GitHub Desktop
2. انقر "Sign in to GitHub.com"
3. أدخل بيانات GitHub
4. وافق على الأذونات
```

#### الخطوة 3: استنسخ المستودع

```
1. انقر "File" > "Clone Repository"
2. اختر "taj-guides"
3. اختر مجلد على جهازك
4. انقر "Clone"
```

#### الخطوة 4: انسخ ملفات الموقع

```
1. افتح مجلد المستودع
2. انسخ جميع ملفات الموقع إليه:
   - index.html
   - styles.css
   - script.js
   - guides/ (المجلد)
   - جميع الملفات الأخرى
```

#### الخطوة 5: رفع الملفات

```
1. في GitHub Desktop، ستجد الملفات الجديدة
2. أدخل رسالة: "Initial commit: Taj Guides Website"
3. انقر "Commit to main"
4. انقر "Push origin"
5. تم! ✅ الملفات مرفوعة
```

### الطريقة 2: استخدام سطر الأوامر

```bash
# الذهاب إلى مجلد المشروع
cd /home/ubuntu/taj-guides-website

# تهيئة Git
git init

# إضافة جميع الملفات
git add .

# إنشاء commit
git commit -m "Initial commit: Taj Guides Website"

# إضافة المستودع البعيد
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/taj-guides.git

# رفع الملفات
git push -u origin main
```

**ملاحظة:** استبدل `YOUR_USERNAME` باسم مستخدمك على GitHub

## المرحلة الرابعة: ربط Vercel مع GitHub

### الخطوة 1: إنشاء حساب Vercel

```
1. اذهب إلى https://vercel.com
2. انقر "Sign Up"
3. اختر "Continue with GitHub"
4. وافق على الأذونات
5. تم! ✅ حسابك مفعّل
```

### الخطوة 2: استيراد المشروع

```
1. اذهب إلى Vercel Dashboard
2. انقر "Add New"
3. اختر "Project"
4. ابحث عن "taj-guides"
5. انقر "Import"
```

### الخطوة 3: إعدادات المشروع

```
الإعدادات الافتراضية كافية:
- Framework: Other (Static)
- Root Directory: ./
- Build Command: (اترك فارغاً)
- Output Directory: (اترك فارغاً)
- Environment Variables: (لا توجد)
```

### الخطوة 4: النشر

```
1. انقر "Deploy"
2. انتظر حتى ينتهي النشر (1-2 دقيقة)
3. ستحصل على رابط مثل: taj-guides.vercel.app
4. تم! ✅ الموقع منشور
```

## 🔧 استكشاف الأخطاء

### المشكلة: الملفات لا تظهر في GitHub

**الحل:**
```
1. تأكد من أنك في المجلد الصحيح
2. تأكد من أن الملفات موجودة
3. جرب الأمر: git status
4. تأكد من أن الـ commit نجح
```

### المشكلة: الموقع لا يحمّل

**الحل:**
```
1. تأكد من أن النشر انتهى
2. امسح ذاكرة التخزين المؤقتة
3. جرب متصفح مختلف
4. تحقق من الأخطاء في Vercel Dashboard
```

### المشكلة: الصور لا تظهر

**الحل:**
```
1. تأكد من أن الصور مرفوعة إلى GitHub
2. تحقق من مسارات الصور في HTML
3. تأكد من أن أسماء الملفات صحيحة
```
